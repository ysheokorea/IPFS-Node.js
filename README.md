# IPFS Node.js 사용법

## ipfs-http-client

- version : 49.0.4

## 실행 방법

1. ipfs cli를 설치한다.
2. `ipfs daemon` 명령어로 IPFS 노드를 실행한다.
3. express에서 IPFS로 파일을 업로드 한다.
4. 해시 반환값을 받아 렌더링 한다.
